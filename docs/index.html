Como funciona / para os técnicos

Este painel conecta por MQTT sobre WebSocket Seguro ao broker público wss://broker.hivemq.com:8884/mqtt. Os tópicos usam o prefixo desta sala: .

  • Simular ensaio: publica uma curva de ruptura em readings/<device> — exatamente o que o gateway ESP32 fará lendo a prensa.
  • Pedir leitura: publica READ? em commands/<device>; o dispositivo (aqui simulado por esta aba) responde com uma leitura — prova o fluxo nuvem→gateway.
  • Troque de sala adicionando ?sala=meulab na URL para isolar seu tráfego.

Repositório e estudo técnico: github.com/SamuelAzo/lab-concreto-gateway
